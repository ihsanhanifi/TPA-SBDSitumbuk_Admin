  1. Buka Google Spreadsheet baru → beri nama "Data Santri TPA Surau Balai Diateh"
  2. Klik Extensions → Apps Script
  3. Paste kode di bawah, lalu klik Deploy → New Deployment → Web App
  4. Set "Who has access" ke Anyone, lalu Deploy
  5. Salin URL Web App dan paste di kolom di bawah
